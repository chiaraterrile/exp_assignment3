\doxysubsection{Detailed Description}
\hypertarget{namespacestate__machine_Description}{}\doxysubsection{Description}\label{namespacestate__machine_Description}
This scripts is a R\+OS node that implements the object detection of six different colors (red,blue,green,yellow,black and magenta) The first time this colored object have been detected, the robot moves towards them and sends informations about their positions and colors to a topic.
