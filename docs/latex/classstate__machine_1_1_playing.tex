\mbox{\hyperlink{classstate__machine_1_1_playing}{Playing}} state execution. 

\hypertarget{namespacestate__machine_Description}{}\doxysubsection{Description}\label{namespacestate__machine_Description}
In this state the robot goes to the user position and waits for a \mbox{\hyperlink{namespace_go_to}{Go\+To}} command by the user. It waits the command for a certain amount of time, if it is not received exits from the P\+L\+AY state. When a \mbox{\hyperlink{namespace_go_to}{Go\+To}} command is received, is checked if the location in the command is known. If it is known, the robot simply goes there, and then come back to the play state. ( so back to the user waiting for another \mbox{\hyperlink{namespace_go_to}{Go\+To}} command). If the location is unknown, the robot switches to the F\+I\+ND behaviour.

\begin{DoxyReturn}{Returns}
\mbox{\hyperlink{classstate__machine_1_1_normal}{Normal}} in case of absence of \mbox{\hyperlink{namespace_go_to}{Go\+To}} commands 

Play state when the desired location has been found 

\mbox{\hyperlink{classstate__machine_1_1_find}{Find}} state when the location is unknown 
\end{DoxyReturn}
